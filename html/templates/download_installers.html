Download installers

Installers are provided for Android, Linux, Mac OS X, and Windows. Please note that Sensibility Testbed only supports sensor access on Android at the moment.


Install via
Google Play

Android instructions

  1. Open this web page on your Android device.
  2. Install SL4A which is a library that Sensibility Testbed requires. SL4A is available from Google Code.
  3. If your device is running a version of Android below 5 ("Lollipop"), tap the "Android" icon to the left to download Sensibility Testbed via Google Play. (If that doesn't work, please try this alternative link.)
  4. If you are running Android 5 ("Lollipop") or 6 ("Marshmallow"), please use our updated BETA installer!
  5. In the Google Play app, tap Install and review the requested permissions.
  6. Start the app and follow the on-screen instructions to complete the installation. Depending on the speed of your device, the process can take a few minutes to finish.

To check that Sensibility Testbed is running, look at the Sensibility Testbed application window.
NoteSensibility Testbed currently does not support sensor access on the platforms listed below!
You may still install Sensibility Testbed to run non-sensor related code.

Download installer
for Linux

Linux instructions

  1. Download the Sensibility Testbed tarball to your computer.
  2. Extract the tarball to the directory in which you would like to install.
  3. Navigate to that directory and run ./install.sh

To check that Sensibility Testbed is running, try running:
ps -ef | grep nmmain.py | grep -v grep

Note: You must have Python 2.7 installed.

Download installer
for Mac

Mac instructions

  1. Download the Sensibility Testbed tarball to your computer.
  2. Extract the tarball to the directory in which you would like to install.
  3. Navigate to that directory and run ./install.sh

To check that Sensibility Testbed is running, try running:
ps Aww | grep nmmain.py | grep -v grep

Note: You must have Python 2.7 installed.

Download installer
for Windows

Windows instructions

  1. Download the Sensibility ZIP file to your computer.
  2. Extract the ZIP file to the directory in which you would like to install Sensibility Testbed.
  3. Double-click install.bat or run it from the command line.

To check that Sensibility Testbed is running, look in Task Manager for the process pythonw.exe.
